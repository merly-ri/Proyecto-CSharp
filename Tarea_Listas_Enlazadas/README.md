# Tarea Listas Enlazadas

## Descripción
Este proyecto corresponde a la tarea sobre listas enlazadas en el lenguaje C#.  
Se implementa una estructura de datos basada en nodos, aplicando conceptos básicos
de Programación Orientada a Objetos.

## Ejercicios realizados
1. Contar el número de elementos de una lista enlazada.
2. Invertir una lista enlazada.

## Archivos del proyecto
- Nodo.cs: Define la estructura del nodo.
- ListaEnlazada.cs: Contiene los métodos de la lista enlazada.
- Program.cs: Ejecuta y prueba los ejercicios.

## Lenguaje
- C#
